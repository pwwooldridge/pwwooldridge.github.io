Handy command for manipulating images in numpy

Posted on Tue 03 September 2019 in misc

When loading multiple images into numpy arrays, usually an array of dimension:

(num_images, num_pixels, num_pixels, 3)

will be created. Quite often, for machine learning purposes, we need to reshape this into an array with dimensions:

(num_images, num_pixels * num_pixels * 3)

This can be achieved with the following one-liner

images_arr.reshape(images_arr.shape[0], -1)

Which keeps returns an array with the first dimension the same as before and all remaining dimensions unravelled into a single dimension.
