Handy commands for manipulating images in Python

Posted on Tue 03 September 2019 in Python

In this post I show some handy commands for working with image data in Python. The following imports are useful when working with images:

import cv2
import glob
import numpy as np

1. Read multiple image files in a directory

images = [cv2.imread(file) for file in glob.glob("images/*.jpg")]

2. Flatten an image to a single dimension for machine learning

load image

im = cv2.imread("images/cat1.jpg")

check dimension of loaded image

im.shape

(1600, 1200, 3)

reshape to a vector of dimension (1, 1600 * 1200 * 3)

flat_im = im.reshape(1, -1)
assert flat_im.shape == (1, 1600 * 1200 * 3)
