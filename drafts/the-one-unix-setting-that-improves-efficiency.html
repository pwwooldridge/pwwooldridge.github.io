The one unix setting that improves efficiency

Anyone who spends a lot of time on the command line will know that often you're looking for a previous command to reuse or repurpose. The following configuration allows me to easily search through their bash history using the up and down arrows. Instead of searching a substring anywhere in the command it will search the history with the text to the left of the cursor. This aspect I find especially useful. To set it up add the following lines to ~/.inputrc:

.. code-block:: sh

"\e[A": history-search-backward
"\e[B": history-search-forward
